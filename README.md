# Telegram Booking Bot (aiogram 3)

Simple Telegram booking bot template built with Python and aiogram 3.

This bot allows clients to:
- Choose a service
- Select available time
- Enter name and phone number
- Send booking request directly to admin

Perfect for small businesses:
- Beauty salons
- Barbershops
- Tutors
- Massage therapists
- Service providers

---

## Features

- Finite State Machine (FSM)
- Configurable services and time slots
- Phone validation
- Admin notifications
- Clean project structure

---

## Project Structure

```
bot.py
config.py
keyboards.py
requirements.txt
```

---

## Installation

1. Clone repository:

```
git clone https://github.com/yourusername/telegram-booking-bot.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Edit `config.py`:
- Insert your bot token
- Insert your Telegram ID
- Customize business name and services

4. Run:

```
python bot.py
```
